\# Quantum Coin Toss Simulator



\## Overview



This project demonstrates a simple quantum coin toss using Qiskit. A qubit is placed into a superposition state using the Hadamard gate and then measured to simulate a fair coin toss.



\## Technologies Used



\* Python

\* Qiskit

\* Qiskit Aer



\## Project Files



\* quantum\_coin\_toss.py

\* draw\_circuit.py



\## Results



The simulator was executed for 1000 shots and produced approximately 50% Heads and 50% Tails, demonstrating quantum randomness.
